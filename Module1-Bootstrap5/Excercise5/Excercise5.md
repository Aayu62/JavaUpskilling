# Exercise 5 Notes

## 5.1 Centering content
`justify-content-center` controls horizontal alignment inside the row, while `align-items-center` controls vertical alignment.
The example uses a roomy container so you can clearly see how the centered card behaves on different viewport sizes.

## 5.2 Reordering columns
`order-md-2` and `order-md-1` change the visual order only on medium screens and larger.
This is useful when you want a mobile-first reading order, but a different desktop layout.

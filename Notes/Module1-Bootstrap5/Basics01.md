🌐 What is Bootstrap 5?
Bootstrap is a front‑end framework — basically a big collection of ready‑made CSS (styling rules) and JavaScript (interactive behaviors).

It helps you quickly build modern, responsive websites without writing all the CSS and JS from scratch.

Version 5 is the latest major release, with updated features like improved grid system, utilities, and no dependency on jQuery.

👉 Think of Bootstrap as a toolbox: instead of hammering out every nail yourself, you grab pre‑built tools (buttons, navbars, grids, forms) and assemble them into a polished site.

🚚 What is a CDN?
CDN stands for Content Delivery Network.

It’s a network of servers around the world that host files (like Bootstrap’s CSS and JS).

When you link Bootstrap via CDN, your browser fetches those files from the nearest server — making your site load faster.

Instead of downloading Bootstrap and keeping it in your project folder, you just paste a link to the CDN in your HTML.

👉 Imagine you want to use a library book.

Local copy = you buy the book and keep it at home.

CDN = you borrow it instantly from the nearest library branch whenever you need it.
